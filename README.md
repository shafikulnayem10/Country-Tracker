
### 🔷 **Project Name**

**Country React App**


### 📝 **Description**

A responsive React-based web application that fetches and displays real-time data on countries from a public API. Users can:

* 🌍 Browse all countries with flags, capitals, and names
* ✅ Mark countries as "Visited" or "Unvisited" (toggle system)
* 🔎 Instantly search countries by name
* 🧠 Built using  **React Hooks**, and modern JSX
* 📦 Deployed to GitHub Pages using `gh-pages`

**Tech Stack:** React, Vite, JavaScript, HTML, CSS, GitHub Pages
**Live Demo:** https://shafikulnayem10.github.io/Country_React_App
